# Grinder

Network for BITS Pilani students. Connect with seniors, find mentors, and join clubs that match your interests.

## Stack

- Vite
- React
- Supabase
- Zustand
- Framer Motion

## Setup

```sh
npm install
npm run dev
```

Runs on `http://localhost:8080`.

## Scripts

- `npm run dev` – dev server
- `npm run build` – production build
- `npm run preview` – preview production build
- `npm run lint` – ESLint
- `npm run test` – Vitest

## Env

Create `.env` with:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_PUBLISHABLE_KEY=...
```
